# Astro UX Lab 2026

Landing page avanzada en Astro que demuestra arquitectura moderna para sitios estáticos con navegación animada, rutas dinámicas pre-renderizadas y tipado estricto.

## Qué incluye

- View Transitions y name transitions.
- 151 páginas estáticas generadas en build time.
- Ruta dinámica con `Astro.params.slug`.
- Componentes reutilizables con props tipadas.
- Layout base con meta tags SEO/Open Graph/Twitter.
- Página `404` personalizada.
- TypeScript path aliases.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run check
```
